## 🚀 Overview
**ADDER** is a sophisticated, high-fidelity dashboard designed with a primary mission: empowering users to effortlessly build and manage their own product catalogs. It provides a seamless, professional interface for users to list, categorize, and track inventory with absolute precision.

---

## ✨ Key Features

### 🛠️ Core Functionality
* **Secure Access:** Integrated password-gate system (`Default: 123`) to protect administrative actions.
* **Smart Product Listing:** Comprehensive form capturing Name, Type, Model, Description, Color, Amount, and Pricing.
* **Real-time CRUD:** Instant addition and deletion of products with immediate UI updates.
* **Persistence:** Fully integrated with `LocalStorage` to ensure data survives page refreshes.

### 🎨 Cinematic UI/UX
* **Proximity Scaling:** Images feature an `overflow-hidden` hover effect for a professional look.
* **Dynamic Counters:** Interactive badges for Shopping Cart, Favorites, and Saved items.
* **Sleek Animations:** Custom CSS keyframes for "Loading States" and "Modal Transitions."
* **Dark Mode Aesthetic:** A premium black and gold (`#d0b57d`) theme.

### ⚙️ Technical Excellence
* **Base64 Image Encoding:** Converts uploaded files to strings for stable local storage.
* **Event Delegation:** Optimized DOM event handling for high performance.
* **Type Safety:** 100% TypeScript implementation for predictable and bug-free code.
